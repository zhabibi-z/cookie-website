# Kevin Cookie Company — Website

A simple static website for the Kevin Cookie Company, handcrafted with plain HTML.

| File | Description |
|---|---|
| `index.htm` | Landing page — company story and values |
| `secret-recipe.htm` | Flagship chocolate-chip cookie recipe |

## Assets

| File | Description |
|---|---|
| `assets/Primary Logo.png` | Official company logo |

## Running locally

No build step required — open any `.htm` file directly in a browser:

```bash
open index.htm
```

Or serve with Python's built-in HTTP server:

```bash
python -m http.server 8080
# then visit http://localhost:8080
```
